# Afrosmoky Guest Seater

Afrosmoky Guest Seater is a web application designed to help event organizers create and manage seating arrangements for events such as weddings, conferences, and banquets. The application provides an intuitive interface for designing seating layouts, assigning guests to seats, and generating printable seating charts.

## Features

- **SVG-based Seating Layouts**: Create and customize seating arrangements using scalable vector graphics (SVG) for high-quality visuals.
- **Guest Management**: Import and manage guest lists, including dietary preferences and special requirements.
- **Interactive Seat Assignment**: Drag and drop guests to assign them to specific seats in the layout.
- **Table Configurations**: Support for various table shapes and sizes, including round, rectangular, T and U shapes.
- **Real-time Collaboration**: Multiple users can work on the same seating plan simultaneously.
- **Export Options**: Export seating charts in various formats, including PDF and image files.

## Technologies Used

- SolidJS for building the user interface.
- SVG for rendering seating layouts.
- TypeScript for type-safe development.
- Vite for fast development and build processes.

## Getting Started

To get started with Afrosmoky Guest Seater, follow these steps:

1. Clone the repository:
  ```bash
    git clone https://github.com/Afrosmoky/event-creator.git
    cd event-creator
  ```
2. Install dependencies:
  ```bash
    npm install
  ```
3. Run the development server:
  ```bash
    npm run dev
  ```
4. Open your browser and navigate to `http://localhost:5147/svg-editor/` to access the application.
